### Step 1: Set Up Gmail App Password

1. **Go to your Gmail account**: https://myaccount.google.com/
2. **Enable 2-Factor Authentication** (required for app passwords)
3. **Generate App Password**:
   - Go to: https://myaccount.google.com/apppasswords
   - Select "Mail" and your device
   - Click "Generate"
   - Copy the 16-character password

### Step 2: Update .env File

Open `Backend/.env` and replace these values:

```env
EMAIL_USER=your-actual-email@gmail.com
EMAIL_PASSWORD=your-16-character-app-password
```

**Example:**
```env
EMAIL_USER=dakshaa2025@gmail.com
EMAIL_PASSWORD=abcd efgh ijkl mnop
```

### Step 3: Restart Backend Server

```powershell
cd Backend
npm start
```

### Step 4: Test Registration

1. Go to signup page
2. Fill in the registration form
3. Submit the form
4. User should receive a welcome email within seconds

## âš ï¸ Troubleshooting

### Email Not Sending?

1. **Check .env credentials**
   - Make sure EMAIL_USER is correct
   - Make sure EMAIL_PASSWORD is the App Password (not your regular password)

2. **Check console logs**
   - Backend console will show "Welcome email sent: <messageId>"
   - Or error message if failed

3. **Check spam folder**
   - First emails might go to spam

4. **Gmail Security**
   - Make sure "Less secure app access" is OFF (use App Password instead)
   - Make sure 2FA is enabled

### Common Errors:

**Error: Invalid login**
- Solution: Generate new App Password

**Error: Connection timeout**
- Solution: Check internet connection

**Error: Daily sending quota exceeded**
- Solution: Wait 24 hours (Gmail free limit: 500 emails/day)
